Rumus Throughput
T = Total Bytes Time Span × 8 ÷ 1000 Kbits/s
Total Bytes
Total ukuran semua paket yang tertangkap
Statistics → Capture File Properties → Bytes (Captured)
Time Span
Selisih waktu antara paket pertama dan paket terakhir
Statistics → Capture File Properties → Time span, s
Rumus Delay
D = Total Delay Jumlah Paket − 1 × 1000 ms
Total Delay
Jumlah seluruh selisih waktu antar paket (delta time). Dihitung otomatis jika upload CSV.
Excel: SUM kolom Delay (Time 2 − Time 1)
Jumlah Paket
Total paket yang tertangkap Wireshark. Dibagi N−1 karena selisih antar N paket menghasilkan N−1 interval.
Statistics → Capture File Properties → Packets (Captured)
Rumus Jitter
J = Total Jitter Jumlah Paket − 1 × 1000 ms
Total Jitter
Jumlah seluruh variasi antar delay. Dihitung sebagai Delay 2 − Delay 1, bisa bernilai positif maupun negatif.
Excel: SUM kolom Jitter (Delay 2 − Delay 1)
Jumlah Paket − 1
Sama dengan delay, variasi dihitung antar pasangan sehingga menghasilkan N−1 data jitter.
Rumus Packet Loss
PL = Paket Hilang Paket Terkirim × 100 %
Paket Terkirim
Total seluruh paket yang tertangkap oleh Wireshark
Statistics → Capture File Properties → Packets (Captured)
Paket Hilang
Jumlah paket yang terdeteksi hilang oleh analisis TCP Wireshark
Filter: tcp.analysis.lost_segment → lihat angka Displayed
